RTF Format
==========

Main Entrance
-------------
Total Height = 24 pt
i.e. 3 lines => line height = 24 / 3 = 8 pt

Font Color:
    Title: #ff0000 (red)
    Body: #ffffff (white)

Building Side
-------------
Total Height = 48 pt
i.e. 3 lines => line height = 48 / 3 = 16 pt

Font Color:
    Title: #00ff00 (green)
    Body: #ffff00 (yellow)
